# Upvote Technical Challenge

This repository contains the code for the Upvote Technical Challenge.

## Live Demo

https://thminggg.github.io/scrawlr-upvote/

## Getting Started

Install npm packages:

```bash
yarn
```

Run the development server:

```bash
yarn dev
```

## Testing

Run the test:

```bash
yarn test
```
